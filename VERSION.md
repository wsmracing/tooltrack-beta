# ToolTrack V4.7.0

Mobile, visual and checkout reliability release:

- Replaced the two-menu mobile header with one role-aware account menu.
- Added mobile Log out and Check serial navigation.
- Kept team controls hidden unless the account has Team or Fleet access.
- Improved dark-mode card, form, result and asset contrast.
- Replaced the cartoon-style TT badge with a cleaner security/technology mark.
- Added a three-image homepage hero rotation with a 10-second interval and smooth fade.
- Disabled PWA installation prompts during the private beta.
- Prefilled checkout contact and saved delivery details from the signed-in account.
- Added optional saving of edited delivery details back to the account.
- Required a verified email before an order can be placed.
- Added basic phone validation and Irish-number normalisation.
- Added an atomic Supabase checkout function to prevent half-created orders.
- Configured Resend examples for noreply@mail.tooltrack.ie with replies to support@tooltrack.ie.
